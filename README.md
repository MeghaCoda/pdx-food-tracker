# About PDX Food Tracker

This site is intended to be a community resource for people seeking free and discounted food in the Portland, Oregon area. I intend for it to become a community project where community members can contribute and verify information as information evolves.


This site is built in React anad Next using Vite and is deployed using vercel and supabase.

Maps are powered by [React Leaflet](https://react-leaflet.js.org/) to avoid the costs involved with google maps usage should this site start getting substantial traffic.
